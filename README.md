# TicketSwift ✈️🚆🚌

TicketSwift is an AI-powered travel ticket companion designed to help users find the most affordable and reliable transportation options across multiple platforms.

## 🚀 Overview
TicketSwift simplifies travel planning by comparing ticket prices from various transportation modes and suggesting the lowest and most trusted booking options.

## ✨ Features
- 🔍 Smart comparison of ticket prices
- 💰 Identifies the most cost-effective travel options
- ✅ Focus on trusted and reliable reservations
- 🤖 AI-driven recommendations
- 🌐 User-friendly interface

## 🧠 How It Works
The chatbot analyzes user queries and retrieves relevant ticket information, helping users choose the best travel option based on price and reliability.
